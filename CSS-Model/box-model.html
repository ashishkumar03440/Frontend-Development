Box Model Exploration

Understanding content, padding, border, and margin visually.

Default Box Model (content-box)

Content Area

In the default model, the total width and height are calculated by adding content + padding + border + margin. This can make precise sizing harder to manage.

Border-box Model (box-sizing: border-box)

Content Area

In border-box, padding and border are included within the element’s total width and height — making layouts more predictable.
